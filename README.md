# ArtWeb
Website for artists

Framworks used:   
-> React  
-> Webpack  
-> Next.js  
-> Firebase  

To run app:  
1) Clone repo  
2) Install npm  
3) Run these commands:  
<code> 
    $cd art-web  
    $npm run dev
</code> 
4) Go to http://localhost:3000
